## 5. 根因分析（Root Cause）

### 5.1 直接原因（Technical Trigger）

- 例如：配置错误、迁移缺失、第三方 API 变更、代码缺陷、容量不足等

### 5.2 深层原因（Systemic Cause）

- 例如：发布流程缺失、监控不全、测试覆盖不足、权限边界不清、文档缺失等

### 5.3 5 Whys（至少 5 层）

1. 为什么发生？
2. 为什么会进入线上？
3. 为什么没有提前发现？
4. 为什么没有自动保护机制？
5. 为什么流程/规范未覆盖？
