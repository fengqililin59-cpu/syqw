# ZhiFlow · 私域增长平台

> 帮中小企业把微信客户变成持续复购的资产

## 一句话定位

ZhiFlow 是面向中小企业的企业微信私域管理系统，
把客户获取、培育、转化、复购四个环节
串成一个自动化闭环，让销售团队少做重复工作、
多做高价值跟进。

## 解决什么问题

| 痛点 | 没有 ZhiFlow | 有 ZhiFlow |
|------|-------------|------------|
| 客户跟进靠记忆 | 漏跟、忘跟、撞单 | 系统自动提醒，意向上升立即预警 |
| 销售离职带走客户 | 客户资产归个人 | 一键继承，资产归企业 |
| 群发消息效率低 | 手动逐个发，费时费力 | 按标签批量群发，定时自动触达 |
| 不知道谁最可能成交 | 凭感觉判断 | AI 意向评分，高意向客户自动浮现 |
| 多渠道触达割裂 | 企微/电话/短信各管各 | 统一管理，触达记录全部归档 |

## 核心能力

- 🎯 **AI 意向评分** — 自动识别高意向客户，销售只跟进值得跟进的人
- ⚡ **意向预警** — 客户意向上升时自动通知销售 + 推荐话术
- 📥 **统一收件箱** — 企微会话集中处理，AI 草稿 + **人工审核后再发**
- 📚 **知识库 + AI 审核台** — 企业话术沉淀，降低乱答与合规风险
- 🎫 **服务台** — 订单与服务工单，收件箱一键转工单
- 📱 **多渠道触达** — 企微消息、电话外呼、短信营销统一管理
- 🤖 **自动化流程** — 新客欢迎、定时跟进、生日问候全部自动执行
- 👥 **团队管理** — 细粒度权限控制，审计日志，离职客户一键继承
- 📊 **数据仪表盘** — 客户增长、转化漏斗、渠道 ROI 一屏掌握

## 技术基础

- 基于企业微信官方 API，合规稳定
- 支持渠道活码、消息回调、会话同步
- 数据存储在客户自己的服务器（可私有化部署）

## 落地页访问

落地页文件：frontend/public/landing.html  
访问地址：https://your-domain.com/landing.html

Nginx 配置（已有 server 块内追加）：

```nginx
location = /landing.html {
  root /path/to/frontend/dist;
  try_files $uri =404;
}
```
