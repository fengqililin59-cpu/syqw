# ZhiFlow 销售话术 · 飞书多维表格导入版

> 基于 [销售一页纸](./competitive-sales-one-pager-zh.md) 拆成「客户类型 × 推荐话术」行表，便于飞书 Bitable 维护、筛选与协作。  
> **资料包入口**：[销售资料包总索引](./sales-kit-README-zh.md) · **演示前勾选**：[演示前检查清单（可打印）](./demo-pre-check-checklist-zh.md)  
> **导入方式**：全选下表 → 复制 → 飞书多维表格新建表 → 首行粘贴（或 CSV 导入）；列名与首行一致即可自动分列。

---

## 字段说明

| 列名 | 用途 |
|------|------|
| 客户类型 | 售前识别的主场景标签 |
| 典型痛点 | 对方亲口说的 1–2 个痛点 |
| 对标竞品 | 客户常拿来比的工具 |
| 推荐话术(30s) | 电梯稿，约 30 秒可读 |
| 演示重点 | 5 分钟演示优先打开的模块 |
| 慎说 | 不能承诺或需诚实说明的点 |
| 定价锚点 | 报价与试用口径 |
