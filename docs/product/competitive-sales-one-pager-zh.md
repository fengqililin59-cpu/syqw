## 1. 一句话 + 目标客户

**ZhiFlow：企微里的销售副驾驶——谁该跟、怎么说、你来发。**

**目标客户**：10–30 人销售 + 投流团队（B2B / 顾问式 / 教培美业获客），要「高意向不漏跟、AI 敢开、投流能算账」，不是功能表最长。

## 4. 竞品话术（各 3 条）

### vs 尘峰

1. 尘峰适合**大团队 + 存档合规 + 营销全家桶**；我们适合 **10–30 人**，要 **意向预警 + AI 草稿人发** 两周见效。  
2. **月费透明、14 天全功能试用**；不必先走长商务盘。  
3. 已有尘峰可谈 **收件箱 AI、意向分、广告 ROI** 作补充，不必推倒重来。

### vs 微伴 / 微盛

1. 微伴功能全；我们做 **销售每日闭环**：待跟进 → AI 话术 → 人发 → 看板，少迷路。  
2. AI **默认草稿 + 人审**，公域只草稿，适合老板对 AI 客服有顾虑的团队。  
3. 诚实：群发模板库、存档深度我们在追；**最看重取证**请以尘峰/微伴为主评。

### vs 有赞 / 微盟

1. 有赞强 **卖货、会员、门店导购**；我们强 **投流获客 + 企微销售转化 + AI 辅助**。  
2. 零售主战场建议有赞；**B2B / 顾问式 / 多渠道线索** 我们更贴。  
3. 用 **本周战果 + 广告 ROI** 14 天验证，不靠功能表长度。

## 5. 常见异议 Q&A（5 条）

| 问 | 答（短） |
|----|----------|
| 你们功能比 XX 少？ | 我们赌三件事：**高意向不漏跟、AI 写你来发、投流能算账**。14 天看战果和 ROI。 |
| AI 会不会乱回客户？ | **默认草稿 + 人审**；半自动仅白名单试点，平台可一键关停。 |
| 生产稳不稳？ | **阶段 A**：与您完成迁移、deep health、权限验收后再扩席；试点单租户优先。 |
| 和扣子什么关系？ | 扣子 **造 Agent**；ZhiFlow **管客户、审核发出、算 ROI**，API 对接即可。 |
| 能不能替代客服部？ | **不能对外承诺全自动**；主打销售副驾驶，省写话术、少漏跟。 |
