# ZhiFlow 功能清单

## 客户管理

| 功能 | 说明 |
|------|------|
| 客户列表 | 搜索、筛选、分页，支持移动端 |
| 客户画像 | 基本信息、标签、阶段、意向分、跟进记录 |
| 阶段管理 | 自定义销售阶段，拖拽推进 |
| 标签管理 | 多维度打标，支持批量操作 |
| 跟进记录 | 手动记录 + 自动归档（通话/短信/企微） |
| AI 意向评分 | 规则评分 70% + AI 评分 30% 混合 |
| 意向预警 | 24 小时内分值上涨 ≥15 分自动通知销售 |
| 批量导入 | Excel/CSV 上传，字段映射，去重预览 |
| 数据导出 | 客户列表、跟进记录 Excel 导出 |
| 软删除 | 误删可恢复，数据不丢失 |

## 企业微信集成

| 功能 | 说明 |
|------|------|
| 扫码登录 | 企微扫码直接登录，无需注册 |
| 客户同步 | 自动同步企微好友到系统 |
| 消息回调 | 客户消息实时入库，用于意向分析 |
| 渠道活码 | 不同来源生成不同活码，精准追踪 |
| 欢迎语自动化 | 新客添加后自动发个性化欢迎语 |
| 离职客户继承 | 员工离职一键转移客户给接替人 |
| 企微侧边栏 | 聊天时直接查客户画像和 AI 话术 |

## 营销工具

| 功能 | 说明 |
|------|------|
| 企微群发 | 文本/图片/链接/小程序，按标签筛选，定时发送 |
| 短信营销 | 单条发送 + 批量群发，支持变量插值，阿里云 SMS |
| 裂变活动 | 任务宝类裂变，邀请计数，奖品发放 |
| 个人微信迁移 | 引导老客户从个人微信迁移到企微 |
| 群管理 + 群 SOP | 同步客户群，定时推送群内容 |
| 营销日历 | 全月推送节点可视化规划 |

## 自动化流程

| 功能 | 说明 |
|------|------|
| 可视化流程编辑器 | 基于 React Flow，拖拽配置 |
| 触发器 | 新客添加、阶段变更、意向分变化、定时等 |
| 条件判断 | 意向分、无回复时长、标签包含等 |
| 执行动作 | 发消息、修改标签、变更阶段、延迟等 |
| 自动化日志 | 每次触发记录可查 |

## 多渠道触达

| 功能 | 说明 |
|------|------|
| 电话外呼 | 一键拨打，手机接听或网页软电话 |
| 通话录音 | 自动存档，系统内播放 |
| 通话记录 | 自动写跟进记录，统计接通率 |
| 短信发送 | 客户详情页直接发单条短信 |
| AI 话术推荐 | 企微侧边栏实时推荐跟进话术 |

## 数据分析

| 功能 | 说明 |
|------|------|
| 数据仪表盘 | 客户增长、成交率、阶段分布 |
| 渠道分析 | 各渠道来源客户数、转化率 |
| 广告 ROI | 广告投放花费与新增客户对比 |
| 意向预警记录 | 所有预警历史，AI 话术查看 |
| 通话统计 | 接通率、平均时长、销售排行 |
| 短信统计 | 发送量、成功率、模板效果 |

## AI 员工（统一收件箱）

| 功能 | 说明 |
|------|------|
| 统一收件箱 | 企微等渠道会话集中查看，支持分配销售阶段 |
| 企微消息同步 | 历史/增量同步客户消息到会话流 |
| 人工回复 | 系统内回复并真实发送到企微客户 |
| AI 回复草稿 | 结合知识库生成草稿，销售审核后再发；可选企微 FAQ/询价自动发（公域仅草稿） |
| AI 审核台 | 待审核列表、通过/驳回、风险分级、自动已发质检 |
| 知识库 | 文档上传、分段存储；**向量检索**（OpenAI Embeddings）+ 关键词兜底 |
| SLA 提醒 | 超时未回复会话提醒负责人（可配置 cron） |
| 公域 Webhook | 抖音/小红书等异构消息入站（需配置密钥） |
| 运营看板 | 待审核数、超时会话、开放工单等概览 |

## 服务台（成交与售后）

| 功能 | 说明 |
|------|------|
| 客户订单 | 手工录入/维护订单，关联客户 |
| 服务工单 | 咨询/退款/投诉等类型，优先级与状态流转；**详情页**处理与结案 |
| 收件箱转工单 | 从会话一键创建工单并关联 thread |
| 权限 | `ticket:*`、`order:*` 独立 RBAC |

## 团队管理

| 功能 | 说明 |
|------|------|
| 多租户 | 每个企业独立数据空间 |
| 角色权限 | 细粒度 RBAC，自定义角色 |
| 审计日志 | 删客户、导出、群发等高危操作留痕 |
| 操作记录 | 所有关键操作可追溯 |

## 系统管理

| 功能 | 说明 |
|------|------|
| 套餐计费 | 免费版/专业版/企业版，用量配额控制 |
| 到期提醒 | 到期前 3 天企微消息提醒 |
| 数据备份 | 支持客户数据批量导出备份 |
