## 🔴 7 大核心差距

### GAP-1: 缺少助贷专业数据模型
**现状**: customers 表是通用 CRM 字段
**建议新字段**:
```sql
ALTER TABLE customers ADD COLUMN credit_status VARCHAR(32) COMMENT '征信状况';
ALTER TABLE customers ADD COLUMN monthly_income DECIMAL(12,2) COMMENT '月收入';
ALTER TABLE customers ADD COLUMN existing_debt DECIMAL(12,2) COMMENT '现有负债';
ALTER TABLE customers ADD COLUMN house_value DECIMAL(12,2) COMMENT '房产估值';
ALTER TABLE customers ADD COLUMN loan_purpose VARCHAR(64) COMMENT '借款用途';
ALTER TABLE customers ADD COLUMN expected_amount DECIMAL(12,2) COMMENT '期望金额';
ALTER TABLE customers ADD COLUMN expected_rate DECIMAL(5,2) COMMENT '期望利率';
```

### GAP-2: 缺少金融产品库 + 智能匹配
**新建表**: `loan_products`, `product_match_logs`
- 银行/机构产品（抵押贷/信用贷/经营贷/装修贷…）
- 产品准入条件配置
- AI 匹配引擎: 客户画像 → Top3 产品 + 通过率 + 预估额度

### GAP-3: 缺少进件报件工作流
**新建表**: `loan_applications`, `loan_materials`
- 收集资料 → 报件到银行 → 审核 → 放款 → 佣金结算
- 银行审批状态追踪

### GAP-4: 缺少佣金/分润系统
**新建表**: `commission_rules`, `commission_records`, `settlement_batches`
- 四层分润：销售 / 团队长 / 渠道 / 公司
- 自动计算 + 审批 + 结算

### GAP-5: 缺少售后-续贷-转介绍闭环
**增强**: service_tickets + 自动回访 SOP
- 放款后 T+30/60/90 自动回访
- 复贷预测 + 还款提醒
- 转介绍激励追踪

### GAP-6: AI 能力浮于表面
**增强**: AI Sales Copilot
- 对话实时话术建议
- 跟进自动总结 + 下次建议
- 基于成功案例的智能推荐

### GAP-7: 渠道获客归因不完整
**增强**: 抖音/快手/小红书/百度 API 数据对接
- CPL/CPA/LTV 实时看板
- 自动关停低 ROI 渠道
